# GameOverseer
A generic game server written in Ruby with assistance from EventMachine.

## Getting started
### Basic getting started

* Clone this repo
* Run `bundle install`
* Edit the config file `/config/config.rb`
* Put services into `/services`
* Run `ruby gameoverseer.rb`
